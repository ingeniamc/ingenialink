IL_EXPORT int il_poller_configure ( il_poller_t *  poller,
unsigned int  t_s,
size_t  buf_sz 
)

Configure poller parameters.

Note
  • The maximum stable polling rate is ~500 Hz (2 ms) for a single register when the servo is enabled.
  • The buffer size must be set according to your application needs. It should be large enough so that it can store all samples collected between subsequent calls to il_poller_data_get.
Parameters
[in]pollerPoller instance.
[in]t_sSampling period (ms).
[in]buf_szBuffer size.
Returns
0 on success, error code otherwise.
Examples:
motion.c.
